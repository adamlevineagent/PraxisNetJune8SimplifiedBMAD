# Praxis Network MVP - Ready for Development

## 🚀 Quick Start

This is the **simplified MVP version** of Praxis Network, ready for immediate development.

### What's Inside
- **7 Must-Have Features** (no more, no less)
- **18 API Endpoints** (down from 35+)
- **Simple Data Model** (7 fields instead of 30+)
- **12-Week Timeline** with clear milestones
- **PostgreSQL Only** (no Redis complexity)

### Development Order
1. Week 1-2: Authentication
2. Week 3-4: Onboarding
3. Week 5-6: Admin Dashboard
4. Week 7-8: Batch Processing
5. Week 9-10: Morning Reports
6. Week 11-12: Introductions

### The Core Concept
Users create AI agents through conversation → Agents network overnight → Users get morning reports → One-click introductions

### What We Cut (On Purpose)
- OAuth/SSO
- Real-time features
- Complex privacy layers
- Mobile app
- Analytics
- Multiple AI models

### Tech Stack
- Backend: NestJS + TypeScript + PostgreSQL
- Frontend: Next.js 14 + Tailwind
- AI: OpenRouter.ai (google/gemini-2.0-flash-exp)
- Email: SendGrid or AWS SES
